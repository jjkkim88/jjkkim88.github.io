## 📜 개요

**록맨 X8(Mega Man X8)**은 캡콤의 록맨 X 시리즈 **8번째이자 마지막 넘버링 타이틀**입니다. 전작 X7이 풀 3D 전환에 실패하며 시리즈 최악의 평가를 받은 후, X8은 **2.5D(2D 게임플레이 + 3D 그래픽)**로 복귀하여 시리즈의 명예를 회복한 작품입니다.

**태그 시스템**으로 X·제로·액슬 중 2명을 선택하여 실시간 교체하며 플레이하고, 각 스테이지마다 독특한 기믹이 적용된 다채로운 레벨 디자인이 특징입니다.

## ⚠️ X7의 실패와 X8의 복귀

X8을 이해하려면 전작 X7의 참사를 알아야 합니다.

| X7의 문제점 | X8의 해결 |
|------------|----------|
| 풀 3D 전환 → 카메라·조작 엉망 | **2.5D 복귀** (2D 게임플레이 + 3D 그래픽) |
| X가 잠금 (보스 8체 격파 또는 레플리로이드 64명 구출 후 해금) | **X 처음부터 사용 가능** |
| 최악의 영어 더빙 ("BURN! BURN! BURN TO THE GROUND!") | **성우 연기 대폭 개선** |
| 느리고 지루한 게임플레이 | **빠른 템포, 자연스러운 움직임** |
| 밋밋한 스테이지 디자인 | **스테이지마다 고유 기믹** |
| 보스전이 단순한 체력 깎기 | 보스전 페이싱 개선 |

## ⚙️ 게임 시스템

### 메탈(Metals) & R&D 연구소

적을 처치하거나 오브젝트를 파괴하면 **메탈(통화)**을 획득합니다. 헌터 베이스의 **R&D 연구소**에서 업그레이드를 구매하는 데 사용합니다.

기존 X 시리즈에서 스테이지 안에 숨겨져 있던 **하트 탱크, 서브 탱크**도 R&D 연구소에서 메탈로 구매하는 방식으로 변경되었습니다.

### X의 아머 시스템

| 아머 | 특징 |
|------|------|
| **이카로스 아머** (빨간 빛) | 공격·방어 특화, 피격 대미지 절반, 넉백 제거 |
| **헤르메스 아머** (파란 빛) | 속도·기동성 특화, 이동·대시 속도 2배, 대시 중 무적 |
| **얼티메이트 아머** (검정+보라) | 히든 아머, 이카로스+헤르메스 능력 통합 |
| **뉴트럴(혼합)** | I·H 파츠 자유 조합 가능 (기가 어택 사용 불가) |

### 리트라이 칩

| 항목 | 설명 |
|------|------|
| 초기 보유 | 2개 |
| 최대 보유 | 5개 |
| 소진 시 | 스테이지 처음부터 재시작 (수집 아이템은 유지) |

실질적인 **난이도 조절 장치**로 기능합니다.

## 📖 스토리

### 배경

매버릭 사건이 끝나지 않는 가운데, 인류는 달 식민지를 위한 **야콥 궤도 엘리베이터** 건설 프로젝트를 시작합니다. 이를 위해 개발된 **신세대 레플리로이드**는 카피 칩을 내장하여 자유롭게 형태를 바꿀 수 있으며, 매버릭화를 방지하는 서브루틴이 탑재되었다고 알려져 있습니다.

야콥 프로젝트 책임자 **루미네(Lumine)**가 바일에게 납치되고, 8체의 신세대 레플리로이드가 매버릭화하면서 사건이 시작됩니다.

### 반전: 루미네의 정체

- 시그마 격파 후, **루미네가 진정한 흑막**임이 드러남
- 모든 신세대 레플리로이드의 카피 칩에 **시그마의 DNA 데이터**가 포함되어 있었음
- 루미네의 주장: 신세대 레플리로이드는 진화의 다음 단계이며, 인간과 구세대 레플리로이드는 도태되어야 한다
- 루미네를 격파하지만, 잔해에서 튀어나온 촉수가 **액슬의 머리 크리스탈을 파손** → 의식 불명
- **미해결 엔딩**: X9가 발매되지 않아 시리즈의 마지막 장면으로 남음

## 🔄 X7 vs X8 상세 비교

| 항목 | X7 (2003) | X8 (2004) |
|------|----------|----------|
| **시점** | 풀 3D (카메라 파탄) | **2.5D** (2D 게임플레이 + 3D 그래픽) |
| **X 해금** | 조건 충족 후 해금 | **처음부터 사용 가능** |
| **게임 속도** | 느림, "빙판 위 스케이팅" 같은 이동 | **빠른 템포, 자연스러운 이동감** |
| **성우** | 최악의 영어 더빙 | **수준급 연기** |
| **스테이지** | 밋밋, 3D 공간에서 적을 무시하고 통과 가능 | **스테이지별 고유 기믹** |
| **보스전** | 데미지 스펀지, 지루함 | 페이싱 개선, 약점 활용 전략 |
| **태그** | 없음 (캐릭터 선택만) | **실시간 교체 + 더블 어택** |
| **메타크리틱** | **58점** | **68점** |
| **유저 평가** | 4.9/10 | 대폭 개선 |

## 📊 평가

| 매체 | 점수 |
|------|------|
| **메타크리틱** | **68점** (Mixed or Average) |
| IGN | 약 7.0 |

> *"올드스쿨 팬들은 기뻐할 것이고, 새로운 세대도 왜 록맨이 대단한지 이해하게 될 것이다"* — Game Informer

**호평**: 2.5D 복귀, 성우 개선, 태그 시스템의 전략성, 스테이지별 다양한 기믹, X7 대비 전방위적 개선

**비판**: SNES 시대 X 시리즈의 정점에는 미치지 못함, 일부 스테이지 기믹의 호불호(옵틱 선플라워 훈련 시뮬레이션), 스토리 전개

### 판매

일본 첫 주 약 15,000장, 총 약 35,500장(일본). 전작 X7보다 평가는 나았으나 상업적 성과는 저조하여, X 시리즈는 이후 휴면 상태에 들어갔습니다.

## 🎨 그래픽·사운드

- **그래픽**: 3D 모델로 그려진 캐릭터와 배경이 2D 게임플레이 위에서 자연스럽게 작동합니다. X7에서 "빙판 위를 미끄러지는 듯"했던 캐릭터 모션이 X8에서는 무게감 있게 개선되었습니다. 스테이지 간 카메라 전환도 IGN이 "seamless"라 평할 정도로 매끄럽습니다.
- **사운드**: 전투 템포를 끌어올리는 록 기반 BGM이 시리즈 전통을 이어갑니다. X7에서 최대 오점이었던 영어 성우 연기가 X8에서 "처음으로 진지하게 좋다고 말할 수 있는 수준"으로 개선되었습니다.

## 🎮 플레이 후기

X7이라는 시리즈 최악의 위기 이후, X8은 "이래서 록맨 X가 록맨 X다"라는 것을 다시 증명한 작품입니다. 2.5D 복귀는 정답이었고, 태그 시스템으로 X·제로·액슬의 개성을 실시간으로 활용하는 재미가 있습니다. 특히 제로의 근접 콤보와 액슬의 카피 변신은 스테이지 재도전 동기를 만들어줍니다.

다만 SNES 시대 X1·X4의 완성도에는 미치지 못합니다. 일부 스테이지 기믹(옵틱 선플라워의 미니게임 연속, 라이드 체이서 강제 구간)은 호불호가 갈리고, 스토리도 루미네의 반전 이후 급하게 마무리되는 느낌입니다. 그리고 이것이 시리즈의 마지막이라는 점이 아쉬움을 더합니다 — 액슬의 크리스탈이 파손된 채 끝나는 미해결 엔딩은 20년이 지난 지금까지도 후속작을 기다리게 만듭니다.
