## 📜 개요

**라쳇 앤 클랭크 트릴로지(Ratchet & Clank Trilogy)**는 **인섬니악 게임즈(Insomniac Games)**가 PS2에서 만든 라쳇 앤 클랭크 시리즈 초기 3부작을 **HD 리마스터**하여 PS Vita 한 장에 담은 컬렉션입니다.

- **라쳇 앤 클랭크** (2002)
- **라쳇 앤 클랭크 2: 고잉 코만도** (2003)
- **라쳇 앤 클랭크 3: 업 유어 아스널** (2004)

"달리고, 쏘고, 모으는" 런앤건 플랫포머의 정수인 3작품을 휴대용으로 즐길 수 있으며, 각 게임마다 **독립된 플래티넘 트로피**가 포함되어 있습니다.

## 🎮 수록 3작품

### 1편: 라쳇 앤 클랭크 (2002)

시리즈의 시작. 벨딘 행성의 롬박스 정비사 **라쳇**과 결함 로봇 **클랭크**가 만나, 은하계를 위협하는 **의장 드렉**을 막기 위해 행성을 돌아다니며 모험합니다.

| 요소 | 내용 |
|------|------|
| **빌런** | 의장 드렉 (Chairman Drek) - 자기 행성을 고의 오염 후 새 행성 건설·판매 |
| **배신자** | 캡틴 쿼크 - 유명 히어로지만 드렉의 광고 모델로 배신 |
| **특징** | 무기/가젯 시스템 확립, 행성 이동 구조 |
| **무기** | 고정 성능 (업그레이드 없음), 스트레이프 미지원 |
| **RYNO** | 150,000 볼트, 시리즈 최강 무기의 시초 |
| **수집** | 골드 볼트 40개, 스킬 포인트 |

### 2편: 고잉 코만도 (Going Commando, 2003)

메가코프 CEO **피즈위젯**에게 고용되어 보곤 은하계에서 도난당한 프로토펫을 추적합니다. 진짜 범인은 다시 한번 **캡틴 쿼크**.

| 요소 | 내용 |
|------|------|
| **최대 혁신** | **무기 경험치 시스템** 도입 - 사용할수록 레벨업·진화 |
| **스트레이프** | 듀얼 스틱 조준·이동 동시 가능 (1편에서 없던 기능) |
| **아머** | 방어구 구매·장착 시스템 추가 |
| **아레나** | 갤럭틱 글래디에이터 - 웨이브 전투 챌린지 |
| **우주 전투** | 전투기 조종 3D 슈팅 섹션 |
| **챌린지 모드** | 회차 플레이(NG+), 메가 버전 무기 해금 |
| **대표 무기** | 쉽이네이터 (적을 양으로 변환), 바운서, RYNO II |

### 3편: 업 유어 아스널 (Up Your Arsenal, 2004)

PS2 트릴로지의 완성작. **닥터 네파리우스**가 바이오블리터레이터로 모든 유기체를 로봇으로 변환하려는 음모를 꾸밉니다.

| 요소 | 내용 |
|------|------|
| **빌런** | 닥터 네파리우스 - 시리즈 대표 악역, 유기체 혐오 로봇 |
| **무기 레벨** | V1~V5 (노멀), V8까지 (챌린지 모드 오메가 버전) |
| **쿼크 비드코믹** | 2D 횡스크롤 미니 스테이지 - 쿼크의 과거 이야기 |
| **가제봇** | 클랭크가 소형 로봇을 지휘하는 퍼즐 섹션 |
| **자이언트 클랭크** | 클랭크 거대화 메카 전투 |
| **아레나** | 어나이얼레이션 네이션 (2편보다 확장) |
| **대표 무기** | 에이전트 오브 둠, 리프트 인듀서, RY3NO, 쿼크-오-레이 (적→오리) |
| **온라인 멀티** | PS2 원작에 포함, **Vita판에서는 삭제** |

## 🔄 PS2 vs PS3 vs PS Vita 비교

| 항목 | PS2 원작 | PS3 HD (2012) | **PS Vita (2014)** |
|------|---------|-------------|-----------------|
| **개발** | Insomniac Games | Idol Minds | Mass Media |
| **해상도** | SD (480i/480p) | **720p HD** | 네이티브 이하 (~640x368 추정) |
| **프레임** | 60fps | **60fps** | **30fps** (간헐적 드롭) |
| **3D 입체** | 없음 | 스테레오스코픽 3D | 없음 |
| **트로피** | 없음 | 게임당 플래티넘 | 게임당 플래티넘 (별도 리스트) |
| **RC3 온라인** | 포함 (서버 종료) | 포함 (2018 서버 종료) | **삭제** |
| **컷씬** | 4:3 네이티브 | 4:3 (좌우 검은 띠) | 4:3 (좌우 검은 띠, 흐릿) |
| **오디오** | 스테레오 | RC2·3 모노 버그 | RC2·3 모노 버그 (동일) |
| **RC1 인풋 랙** | 없음 | 약간 있음 | **심각** |
| **휴대성** | 없음 | 없음 | **O** |
| **메타크리틱** | - | **83점** | **76점** |

### Vita판의 강점

- **3편 풀 게임을 한 카트리지/다운로드로** 40시간 이상의 콘텐츠
- 게임당 **플래티넘 트로피 3개** (PS3과 별도 리스트)
- Jak & Daxter Vita 이식보다 **안정적인 퍼포먼스** (크래시·티어링 없음)
- **2편·3편은 원활하게 구동**, 30fps에서도 플레이에 큰 지장 없음

### Vita판의 약점

- **1편의 심각한 인풋 랙**: Push Square가 "borderline broken"으로 평가. 정밀 플랫포밍 구간에서 조작이 밀리는 현상
- 30fps로 PS3판(60fps) 대비 **절반 프레임**
- **뒷면 터치패드로 스트레이프** 매핑 → 오작동·불편
- 컷씬이 4:3에 저해상도라 **흐릿하게 표시**
- BGM이 간헐적으로 끊기거나 재시작되는 오디오 버그

## 🎨 그래픽·사운드

- **그래픽**: PS2 원작 대비 텍스처가 깨끗해졌고, Vita 화면에서 캐릭터와 이펙트가 선명하게 표시됩니다. 다만 PS3 HD판(720p/60fps)과 비교하면 해상도와 프레임 양쪽에서 열세이며, 4:3 컷씬은 작은 화면에서 흐릿하게 보입니다.
- **사운드**: 인섬니악 특유의 경쾌한 BGM과 타격음·폭발음이 액션의 리듬감을 살려줍니다. RC2·RC3에서 모노 오디오 버그(PS3판에서 물려받은 문제)와 간헐적 BGM 끊김이 있는 점은 아쉽습니다.

## 🎯 추천 포인트

1. PS2 시대 최고의 액션 플랫포머 3부작을 **휴대용 하나로** 즐기고 싶은 경우

4. 2편(고잉 코만도)·3편(업 유어 아스널) 중심으로 플레이할 계획인 경우

## 🎮 플레이 후기

라쳇 앤 클랭크 PS2 3부작은 20년이 지나도 재미있는 게임입니다. 달리면서 쏘고, 볼트를 모아 무기를 사고, 무기가 경험치로 진화하는 루프는 중독성이 있습니다. 특히 2편에서 도입된 무기 레벨업 시스템과 3편의 V5→오메가 진화는 "이 무기를 한 번 더 써봐야지"라는 동기를 끊임없이 만들어줍니다.

Vita판의 문제는 명확합니다. **1편의 인풋 랙은 실제로 체감**됩니다. 정밀 점프가 필요한 구간에서 조작이 밀리는 느낌은 스트레스입니다. 하지만 2편과 3편은 30fps에서도 충분히 쾌적하게 돌아가며, 뒷면 터치패드의 스트레이프 불편함도 익숙해지면 감수할 수 있는 수준입니다.

최고의 경험을 원한다면 PS3 HD 컬렉션(720p/60fps)이 정답이지만, **3편 풀 게임을 들고 다닐 수 있다**는 것 자체가 Vita판의 가치입니다. 1편을 가볍게 넘기고 2·3편에 집중한다면 충분히 만족스러운 휴대용 경험입니다.
