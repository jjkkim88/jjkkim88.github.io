**메다롯트 7**(メダロット7)은 2012년 9월 13일 닌텐도 3DS로 발매된 메다롯트 시리즈의 넘버링 타이틀입니다. 포켓몬처럼 **카부토 버전(カブトVer.)**과 **쿠와가타 버전(クワガタVer.)** 두 가지로 나뉘어 발매되었으며, 시리즈 최초로 **풀 3D 그래픽**을 도입한 작품입니다.

| 항목 | 정보 |
|------|------|
| 발매일 | 2012년 9월 13일 (일본 전용) |
| 개발 | 델타 아츠 (이매지니어 자회사) |
| 발매 | 로켓 컴퍼니 |
| 가격 | 5,800엔 (세별) |
| 패미통 | **30/40** (8/8/7/7) |
| 메다롯트 수 | 140체 |
| 파츠 수 | 700개 이상 |
| 메달 수 | 90종 이상 |
| 해외 발매 | **없음** (일본 전용) |

## 🪲 카부토 vs 쿠와가타 — 두 버전의 차이

메다롯트 시리즈는 1997년 초대작부터 **카부토(장수풍뎅이)**와 **쿠와가타(사슴벌레)** 두 버전으로 발매되어 온 전통이 있습니다. 메다롯트 7도 이 전통을 따릅니다.

### 파트너 메다롯트

두 버전의 가장 큰 차이는 **파트너 메다롯트**(스타터)입니다.

| 항목 | 카부토 버전 | 쿠와가타 버전 |
|------|-------------|---------------|
| 파트너 | **메타비 (メタビー)** | **로크쇼 (ロクショウ)** |
| 형번 | KBT (Kabuto) 타입 | KWG (Kuwagata) 타입 |
| 전투 스타일 | **사격형** — 전 무장이 원거리 사격 | **격투형** — 근접 검/칼날 공격, 고속 회피 |
| 이름 유래 | Metal Beetle (메탈 비틀) | 녹청(緑青, rokusho) |
| 난이도 | **초보 추천** — 원거리 사격이 안정적 | **숙련자용** — 접근 필요, 회피 의존 |

메타비와 로크쇼는 메다롯트 시리즈 전체의 **아이콘 마스코트**입니다. 애니메이션, 만화, 피규어 등 미디어 믹스 전반에서 시리즈를 대표하는 캐릭터입니다.

### 버전별 차이 상세

| 항목 | 카부토 | 쿠와가타 |
|------|--------|----------|
| 파트너 메다롯트 | 메타비 (사격형) | 로크쇼 (격투형) |
| 버전 한정 메다롯트 | 카부토판 전용 일부 기체 | 쿠와가타판 전용 일부 기체 |
| 버전 한정 파츠/메달 | 카부토판에서만 입수 가능 | 쿠와가타판에서만 입수 가능 |
| 특전 AR 카드 | 32종 | 32종 (합계 64종) |
| 스토리 | **동일** | **동일** |
| 전투 체감 | 사격이 안전거리에서 안정적 | 접근전 필요하나 회피 능력 우수 |

스토리는 양 버전 완전 동일합니다. 핵심 차이는 **스타터와 버전 한정 기체/파츠**이며, 전체 도감(140체)을 완성하려면 **버전 간 통신 교환이 필수**입니다.

### 어떤 버전을 골라야 할까?

- **처음 메다롯트를 접한다면 → 카부토 버전**: 메타비의 사격형 무장이 안전거리에서 안정적으로 데미지를 넣어, 초반 난이도가 완만합니다
- **시리즈 경험이 있거나 근접전을 좋아한다면 → 쿠와가타 버전**: 로크쇼의 격투+회피 플레이는 높은 스킬 상한을 가지며, 숙련될수록 강해집니다

## 📖 메다롯트 7 스토리

### 무대

2033년, 메다롯트(소형 파트너 로봇)가 전 세계적으로 보급된 근미래. 조용한 시골 마을 **모로코시 마을**(もろこし町)이 무대입니다.

### 주인공

**아마쿠라 아즈마**(アマクラ アズマ). 메다롯트 DS에 이어 등장하지만, 세계관이 리부트되어 **패러렐 타임라인**입니다. 시리즈 최초의 **중학생 주인공**으로, 이전까지 초등학생이 주인공이었던 전통을 깼습니다.

### 줄거리

1. 평화로운 모로코시 마을에 **수수께끼의 운석**이 떨어짐
2. 사건을 계기로 세계에 불안이 퍼지기 시작
3. 아즈마는 메다롯트 제국의 문을 여는 열쇠 **M 드라이브**를 모으게 됨
4. 시리즈 단골 악당 **로보로보단**의 암약, 아즈마 아버지와 연결된 **M 프로젝트**의 비밀
5. 최종 보스 **헤베레케 박사**와 그의 메다롯트 **G 엠페러** 등장
6. 클라이맥스에서 **달의 낙하**를 막는 전개

### 멀티 엔딩

3명의 히로인 호감도에 따라 **3종류의 엔딩**이 분기합니다:

| 히로인 | 호감도 올리는 법 |
|--------|-----------------|
| 치토세 | 동행 중 "로보틀 함께 하기" 선택 |
| 안 | 동행 중 "로보틀 함께 하기" 선택 |
| 사쿠라 | 사쿠라가 준 파츠를 장착 |

호감도를 올리지 않으면 기본적으로 사쿠라 루트가 됩니다.

## ⚔️ 메다롯트 7 전투 시스템 — 로보틀

### 3대3 로보틀

양쪽 각 **최대 3체의 메다롯트**로 전투합니다. 첫 번째로 선택한 기체가 **리더**가 됩니다.

**승리 조건**: 상대 **리더의 머리 파츠를 파괴**하면 즉시 승리
**시간 초과**: 파괴한 메다롯트 수, 부순 파츠 수, 성공한 공격 수로 판정

### 게이지 기반 액션 시스템

메다롯트의 전투는 일반적인 턴제가 아닌 **리얼타임 게이지 시스템**입니다:

| 요소 | 설명 |
|------|------|
| 커맨드 라인 (파란색) | 명령을 입력하는 대기 라인 |
| 액티브 라인 (빨간색) | 행동을 실행하는 라인 |
| 충전(充填) | 행동 준비 시간 — 파츠 성능에 따라 다름 |
| 냉각(冷却) | 행동 후 쿨다운 — 짧을수록 다음 행동이 빠름 |

다리 파츠의 성능과 지형 궁합에 따라 속도가 크게 달라지며, 빠른 파츠일수록 행동 빈도가 높아집니다.

### 파츠 파괴 시스템 — 전략의 핵심

메다롯트의 4개 파츠는 각각 독립된 장갑(HP)을 가지며, 파괴 시 효과가 다릅니다:

| 파츠 | 파괴 시 효과 |
|------|-------------|
| **머리 (頭部)** | **기능 정지 = 즉시 전투 불능**. 리더라면 패배 |
| **오른팔 (右腕)** | 해당 팔의 공격/기능 상실 |
| **왼팔 (左腕)** | 해당 팔의 공격/기능 상실 |
| **다리 (脚部)** | 모든 능력치 **절반으로 하락** — 치명적 둔화 |

이 시스템이 만드는 전술적 선택지:
- **머리 집중 공격**: 빠른 승리를 노리지만, 머리는 보통 장갑이 높음
- **양팔 파괴**: 상대의 화력을 먼저 제거하는 안전한 전략
- **다리 파괴**: 속도를 절반으로 만들어 행동 횟수에서 압도

### 공격 유형

| 유형 | 설명 |
|------|------|
| 사격 (射撃) | 원거리 공격, 행동 전 타깃 지정 |
| 격투 (格闘) | 근접 공격, 실행 시 가장 가까운 적 공격 |
| 보조 (補助) | 버프/디버프 |
| 회복 (回復) | 아군 파츠 수리 |
| 설치 (設置) | 트랩 배치 |
| 방어 (防御) | 방어 행동 |

### 메다포스 (メダフォース)

차지 게이지가 **100%**에 도달하면 사용 가능한 필살기입니다. 메달마다 배우는 메다포스가 다르며, 강력하지만 차지 중 무방비 상태가 됩니다.

### 메달 시스템

메달은 메다롯트의 **두뇌이자 영혼**입니다.

| 항목 | 설명 |
|------|------|
| 역할 | 메달 없이는 메다롯트가 기동 불가 |
| 성격 | 타겟 선택 행동에 영향 (손상된 부위 우선, 랜덤 등) |
| 궁합 | 파츠와 메달의 궁합이 맞아야 풀 성능 발휘 |
| 스킬 | 3개 스킬 선택 가능 (속도, 파워 등) |
| 성장 | 분기형 진화 (메다롯트 5 이후 부활) |

## 🆕 메다롯트 7 신규 시스템

### 라이선스 시스템 (メダ検定)

메다롯트 7만의 독특한 **단계적 해금 튜토리얼** 시스템입니다:

1. 메다시티의 **메다롯터즈 샵**에서 시험관 **쿠루미**를 만남
2. 기능 설명 → **퀴즈**(4지선다 또는 ○×5문제) → **실전 배틀 테스트**
3. 비용: 500~5,000엔 (첫 번째는 무료, 필수)
4. 총 **8종 라이선스**: 차지 게이지, 맵, 링크, 로테이션(4랭크), 메달리아, 메다포스, 콤보, 메다체인지

스토리 진행에 따라 순차적으로 해금되며, 라이선스를 취득해야 해당 기능을 사용할 수 있습니다.

### 콤보 시스템 (NEW)

**2~3개 파츠를 동시에 사용**하는 합체 공격입니다.

| 장점 | 단점 |
|------|------|
| 한 번에 대량 데미지 | 충전·냉각 시간 대폭 증가 |
| 같은 유형 조합 시 추가 보정 | 실행 전 관련 파츠가 파괴되면 **콤보 전체 실패** |
| CPU 상대로 매우 강력 | PvP에서는 리스크 큼 |

### 메다체인지 개선

**퓨어 세트**(4파츠 전부 같은 기체) 메다롯트가 변신하는 시스템입니다:

| 조건 | 차지 게이지 100% + 모든 파츠 무손상 |
|------|------|
| 변신 중 | HP가 합산, 행동 성능 대폭 변화, 다리 유형 변경 가능 |
| 제한 | 메다포스 사용 불가, 행동마다 차지 게이지 10 소모 |
| 연출 | **풀 3D 변신 애니메이션** (3DS 입체시 지원) |

## 🎮 메다롯트 7 공략 팁

### 1. 리더의 머리를 지켜라

전투의 핵심 규칙입니다. 리더의 머리가 파괴되면 **즉시 패배**합니다. 리더에는 **머리 장갑이 높은 기체**를 배치하고, 방어형 아군으로 리더를 보호하세요.

### 2. 상대 리더의 머리를 노려라

반대로 상대 리더의 머리를 집중 공격하면 빠른 승리가 가능합니다. 다만 머리 장갑이 높은 경우, 팔이나 다리를 먼저 부수는 우회 전략이 효율적일 수 있습니다.

### 3. 다리 파괴는 가성비 최고

다리가 부서지면 **모든 능력치가 절반**이 됩니다. 충전·냉각이 느려져 행동 횟수가 격감하므로, 파워가 강한 적의 다리를 먼저 부수면 전투가 훨씬 수월해집니다.

### 4. 메달-파츠 궁합 확인

메달과 파츠의 궁합이 맞지 않으면 풀 성능이 나오지 않습니다. 새 파츠를 장착할 때 **궁합 표시를 반드시 확인**하세요.

### 5. 팀 구성 밸런스

| 역할 | 추천 구성 |
|------|-----------|
| 공격수 | 사격형 or 격투형 파츠 장비 |
| 서포터 | 회복·보조 파츠로 아군 지원 |
| 탱커 | 탱크 다리 + 높은 장갑으로 리더 보호 |

3체 전부 공격형으로 가면 회복이 안 되고, 전부 서포터면 화력이 부족합니다. **최소 1체는 회복/보조 역할**을 배치하세요.

### 6. 콤보는 CPU전에서, PvP에서는 주의

콤보는 CPU 상대로 매우 강력하지만, 대인전에서는 충전 중 파츠가 파괴당할 리스크가 큽니다. PvP에서는 단일 파츠 공격을 기본으로 하되, 확실한 상황에서만 콤보를 사용하세요.

### 7. 지형에 맞는 다리 선택

전투 필드마다 지형이 다릅니다. 수중 맵에서 이각(二脚)을 쓰면 불리하고, 도로 맵에서 잠수형을 쓰면 낭비입니다. **호버형은 모든 지형에 무난**하므로 지형을 모를 때의 안전 선택지입니다.

### 8. 버전 교환으로 도감 완성

140체 전부를 모으려면 카부토-쿠와가타 간 **통신 교환이 필수**입니다. 주변에 다른 버전 소유자가 있다면 적극 교환하세요.
