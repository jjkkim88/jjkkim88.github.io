**아크 시스템 웍스(Arc System Works)** 개발의 2D 대전 격투 게임으로, 블레이즈블루 시리즈의 **3번째 메인 타이틀**이다. 2012년 아케이드로 먼저 가동되었고, PS3 → PS Vita 순으로 콘솔 이식되었다. 전작 컨티뉴엄 시프트에서 19~20명이던 로스터가 **26명**으로 확대되었고, **오버드라이브(Overdrive)**와 **크러시 트리거(Crush Trigger)** 등 전투 시스템이 대폭 개편되었다. PS Vita 버전은 PS3 버전과 게임플레이·콘텐츠가 거의 동일하며, Vita의 OLED 화면에서 2D 스프라이트가 더 선명하게 보인다는 평가를 받았다.

## 📖 스토리

이카루가 연방을 무대로, 전작 컨티뉴엄 시프트의 사건 이후가 그려진다. 스토리 모드는 3개의 시나리오("크로노판타즈마", "제7기관", "식스 히어로즈")로 구성되어 있으며 **30시간 이상**의 분량이다.

- **라그나 더 블러드엣지**: NOL의 총통(임페라토르)이 자신의 여동생 사야임을 알게 되고, 부활한 숙적 유키 테루미를 쓰러뜨리기 위해 이카루가로 향한다
- **진 키사라기·노엘 버밀리온·마코토 나나야**: 테루미의 세뇌에 빠진 친구 **츠바키 야요이**를 구출하기 위해 움직인다
- **노엘**: "아오의 계승자"로서 마스터 유닛 아마테라스에 접근할 수 있는 능력을 제어하기 위해 레이첼 알루카드를 찾아간다
- 최종 적대자로 유키 테루미, 렐리우스 클로버, **이자나미(임페라토르)**가 등장하며, 식스 히어로즈의 이야기가 이번 작에서 완결된다

치비 캐릭터가 블레이즈블루 세계관을 해설하는 "가르쳐줘 리치 선생" 코너도 수록되어 있어, 시리즈 입문자도 스토리를 따라갈 수 있다.

## 🔄 전작과의 비교

블레이즈블루 시리즈는 매 작품마다 시스템이 크게 바뀌는 것이 특징이다. 크로노 판타즈마는 시리즈 3번째 작품답게 전작들의 시스템을 대폭 개편했다.

### 시리즈 로스터 변천

| 작품 | 연도 | 캐릭터 수 |
|------|------|----------|
| 칼라미티 트리거 (1st) | 2008/2009 | **12명** |
| 컨티뉴엄 시프트 (2nd) | 2010 | ~18명 |
| 컨티뉴엄 시프트 익스텐드 | 2011 | 19~20명 |
| **크로노 판타즈마 (3rd)** | **2012/2013** | **26명** |
| 크로노 판타즈마 익스텐드 | 2014/2015 | 28명 |
| 센트럴 픽션 (4th) | 2015/2016 | 36명 |

### 시스템 변경점 (컨티뉴엄 시프트 → 크로노 판타즈마)

| 항목 | 컨티뉴엄 시프트 | 크로노 판타즈마 |
|------|--------------|--------------|
| 가드 시스템 | **가드 프라이머** (방어 시 스톡 소모, 소진 시 가드 크러시) | 가드 프라이머 **삭제** |
| 가드 브레이크 | 가드 프라이머 소진으로 자동 발생 | **크러시 트리거** (히트 게이지 25% 소비, 능동적 발동) |
| 버스트 | **골드 버스트** (공격·콤보 연장용) | 골드 버스트 **삭제**, **오버드라이브**로 대체 |
| 파워업 | 없음 | **오버드라이브** (전 캐릭터 임시 강화 상태) |
| 버스트 게이지 회복 | 라운드당 고정 횟수 | **시간 경과로 자동 회복** |
| 콤보 | 긴 콤보, 긴 히트스톱 | 히트스톱 감소, 콤보 **짧고 빠르게** 변경 |
| 캐릭터 | ~19명 | **26명** (7명 신규) |

가장 큰 변화는 **오버드라이브 시스템의 도입**과 **가드 프라이머의 삭제**다. 전작에서는 방어만 하다 보면 가드 프라이머가 깎여 가드 크러시가 나는 수동적 시스템이었지만, 크로노 판타즈마에서는 공격 측이 히트 게이지를 소비해 크러시 트리거를 발동하는 능동적 구조로 바뀌었다.

## ⚔️ 전투 시스템

### 기본 조작

A(약), B(중), C(강), D(드라이브)의 **4버튼** 체계. 격투게임 중에서는 버튼 수가 적은 편이라 입문 장벽이 상대적으로 낮다.

### 드라이브 시스템

블레이즈블루의 핵심. 각 캐릭터마다 D 버튼에 **고유한 능력**이 배정되어 있다.

| 캐릭터 | 드라이브 | 효과 |
|--------|---------|------|
| 라그나 | Soul Eater | 공격 시 상대의 체력을 흡수해 회복 |
| 진 | Frost Bite | 공격에 빙결 효과 부여 |
| 노엘 | Chain Revolver | 빠른 연쇄 콤보 |
| 레이첼 | Silpheed | 바람을 조작해 상대/자신을 밀고 당김 |
| 타오카카 | Dancing Edge | 벽과 천장을 자유롭게 이동 |
| 하쿠멘 | Zanshin | 카운터 공격 (상대 기술에 반격) |

### 오버드라이브 (신규)

A+B+C+D 동시 입력으로 발동. **버스트 게이지**를 전부 소비하여 일시적 강화 상태에 돌입한다.

- **지속 시간**: 남은 체력이 적을수록 길어진다 (5~15초)
- **효과**: 드라이브 능력과 필살기가 강화됨
- **발동 시 무적**: 리버설(역전기)로 활용 가능
- **타이머 정지**: 오버드라이브 중에는 라운드 타이머가 멈춤
- 콤보 중 캔슬로 사용 가능 (오버드라이브 캔슬)

### 크러시 트리거 (신규)

A+B를 누르며 **히트 게이지 25%**를 소비. 상대가 배리어 가드를 사용하지 않고 있으면 가드를 부순다. 차지해서 딜레이를 걸면 배리어 게이지에 더 큰 데미지를 준다.

### 히트 게이지 & 필살기

- **히트 게이지**: 4칸(각 25%)으로 구성. 공격 적중·피격 시 축적
- **디스토션 드라이브**: 50% 이상 소비하는 강력한 필살기. 캐릭터당 2~3개
- **아스트랄 히트**: 히트 게이지 100% + 상대 체력 35% 이하 + 매치 포인트 + 버스트 게이지 보유 시 사용 가능한 즉사기. 캐릭터별 화려한 전용 연출

## 🎮 PS Vita 버전의 특징

### PS3 버전과의 비교

| 항목 | PS3 | PS Vita |
|------|-----|---------|
| 게임플레이 | 기준 버전 | **PS3과 동일** |
| 해상도 | 720p / 1080i | **960x544** (Vita 네이티브) |
| 프레임레이트 | 60fps | **60fps** (안정적) |
| 스토리 모드 | O | O (동일) |
| 아케이드/VS/트레이닝 | O | O (동일) |
| 어비스 모드 | O | O (동일) |
| 온라인 대전 | 랭크/플레이어 매치 + **로비 시스템** | 랭크/플레이어 매치 (**로비 미지원**) |
| 로컬 대전 | O | **애드혹** (각자 Vita + 소프트 필요) |
| 크로스 플랫폼 | - | **PS3과 크로스플레이 미지원** |
| OLED 화면 | - | 2D 스프라이트가 **더 선명하고 생생**하게 보임 |

### 조작

Vita에는 L2/R2 버튼이 없기 때문에, FN1·FN2 기능이 **좌우 아날로그 스틱 클릭**에 배정되어 있다. D패드 조작이 기본이며, 격투게임 특성상 D패드가 아날로그 스틱보다 정밀한 입력에 유리하다.

### Vita 버전의 장단점

**장점**: PS3과 동일한 게임플레이·콘텐츠를 휴대용으로 즐길 수 있다. OLED 화면에서 아크 시스템 웍스의 2D 스프라이트가 매우 선명하게 표현된다. 60fps 안정 구동.

**단점**: 온라인 로비 시스템 미지원으로 매칭이 PS3보다 불편하다. PS3과의 크로스플레이가 불가능해 온라인 인구가 제한적이다. 스피커에서 음성·BGM 왜곡이 간헐적으로 발생한다는 리포트가 있다.

## 🎨 그래픽·사운드

### 그래픽

아크 시스템 웍스 특유의 **고해상도 2D 애니메이션 스프라이트**가 핵심이다. 각 캐릭터의 모션이 매우 세밀하게 그려져 있으며, 디스토션 드라이브와 아스트랄 히트 발동 시 삽입되는 컷인 연출이 화려하다. PS Vita의 OLED 화면에서 색감이 더욱 생생하게 보이며, 작은 화면에서도 캐릭터 판별이 명확하다.

### 사운드

이시와타리 다이스케(石渡太輔) 감수의 록/메탈 기반 BGM이 대전의 템포를 끌어올린다. 캐릭터별 전용 테마곡이 있으며, VS 모드에서는 BGM을 자유롭게 선택할 수 있다. 일본어 풀 보이스로 스토리와 전투 대사가 수록되어 있다.

## ⚠️ 문제점

### 스토리 진입 장벽

시리즈 3번째 작품인 만큼, 전작들의 스토리를 모르면 이해하기 어렵다. "가르쳐줘 리치 선생" 코너가 보완해주지만, 근본적으로 시리즈 전체를 관통하는 복잡한 세계관은 입문자에게 부담이 된다.

### DLC 의존

카구라, 테루미, 코코노에 등 콘솔 전용 캐릭터 외에도 추가 캐릭터 DLC(캐릭터당 약 $8)가 있어, 풀 로스터를 갖추려면 추가 비용이 필요하다.

### Vita 온라인 제한

로비 시스템 미지원, PS3 크로스플레이 불가로 온라인 대전 환경이 PS3보다 열악하다. Vita 온라인 인구가 적어 매칭이 어려울 수 있다.

## 🚀 맺음말

블레이즈블루 크로노 판타즈마는 시리즈의 전투 시스템이 가장 세련된 형태로 다듬어진 작품이다. 가드 프라이머를 없애고 오버드라이브·크러시 트리거를 도입한 시스템 개편은 공격과 방어 양쪽에 더 많은 선택지를 부여했다. 26명의 캐릭터 각각이 고유한 드라이브 능력을 가지고 있어, 자신에게 맞는 캐릭터를 찾아 파고드는 재미가 시리즈 최고 수준이다. PS Vita 버전은 PS3과 동일한 게임플레이를 60fps로 구동하며, OLED 화면에서 2D 스프라이트가 더욱 빛을 발한다. 온라인 로비 미지원과 크로스플레이 불가는 아쉽지만, 싱글 콘텐츠만으로도 충분한 볼륨을 제공한다. 휴대용으로 본격 격투게임을 즐기고 싶다면, Vita 라이브러리에서 최고의 선택지 중 하나다.
