**몬스터 헌터 포터블 3rd**(Monster Hunter Portable 3rd, MHP3rd)는 2010년 12월 1일 캡콤이 PSP로 발매한 헌팅 액션 RPG입니다. 발매 2주 만에 **258만 장**, 이후 **480만 장 이상**을 판매해 **PSP 역대 최고 판매작**이자 캡콤 역사상 가장 빠르게 팔린 타이틀이 되었습니다. 패미통 점수 **38/40**을 기록하며 시리즈 최고 평가 중 하나를 받은 작품입니다.

| 항목 | 정보 |
|------|------|
| 발매일 | 2010년 12월 1일 (일본) |
| 개발·발매 | 캡콤 |
| 패미통 | **38/40** |
| PSP 판매량 | **480만 장 이상** (PSP 역대 1위) |
| HD 버전 | PS3 (2011년 8월 25일, 일본) |
| 해외 발매 | **없음** (일본 전용) |
| 전작 | MHP2G (Unite, 2008) |

## 🏯 유쿠모 마을 — 온천과 화지(和紙)의 거점

MHP3rd의 새로운 거점 **유쿠모 마을**(ゆくも村)은 **안개 낀 산속 온천 마을**을 테마로 합니다. 일본식 목조 건축과 화지 분위기가 가득한 아늑한 마을로, 시리즈 거점 중 팬들에게 인기가 높습니다.

### 온천 시스템

전작의 아이루 식당(뷔페) 대신, MHP3rd에서는 **온천**과 **아이루 음료 아줌마**가 밥을 대신합니다:

| 시스템 | 효과 |
|--------|------|
| 온천 (마을 강화) | 체력 +50, 스태미너 +50 (최대 강화 시) |
| 음료 아이루 | 퀘스트 출발 전 스킬 효과 부여 (기존 식당 역할) |

온천은 퀘스트를 반복할수록 레벨이 올라가 최종적으로 체력과 스태미너를 각각 50씩 늘려줍니다. 초반에 온천 레벨을 올리는 것이 체력 관리에 크게 도움이 됩니다.

## 🆚 MHP3rd vs MHP2G (Unite) — 무엇이 달라졌나

| 항목 | MHP2G (2008) | MHP3rd (2010) |
|------|--------------|----------------|
| 거점 | 포코 마을 | **유쿠모 마을** (온천 테마) |
| G랭크 | **있음** | **없음** (HR7까지) |
| 무기 종류 | 11종 | **12종** (스위치 액스 추가) |
| 아이루 동행 | 1마리 | **2마리** 동시 |
| 식당 시스템 | 아이루 식당 (뷔페) | **온천 + 음료 아이루** |
| 신규 대형 몬스터 | — | **17체** (신종 + 아종 포함) |
| MH3 이식 몬스터 | 없음 | **라기아크루스, 바롯사 등** 이식 |
| 퀘스트 총수 | 330+ | 약 230 |
| 설계 방침 | 방대한 볼륨, G랭크까지 | **완성도와 입문성 중시** |

MHP2G가 방대한 볼륨과 G랭크까지 포함한 집대성이었다면, MHP3rd는 **신규 플레이어도 진입하기 쉬운 입문성**과 **신규 컨텐츠의 질적 충실도**를 앞세운 타이틀입니다. G랭크가 없는 것이 단점이지만, 하이랭크(HR7)까지의 완성도와 새로운 몬스터들로 충분히 즐길 수 있습니다.

## 🐉 MHP3rd 몬스터 로스터

총 **60체**(소형 20체 + 대형 40체)가 등장합니다.

### 간판 몬스터 — 진오우거 (ジンオウガ / Zinogre)

MHP3rd의 새로운 얼굴입니다. **번개 속성의 팽룡(牙竜)종**으로, 전기를 머금은 뇌충(雷虫)을 몸에 모아 강화 상태로 변신하는 독특한 메카닉을 가집니다.

- **일반 상태**: 행동이 느리고 공격력이 낮음
- **강화 상태 (번개 충전)**: 전신이 번개로 뒤덮이고 패턴이 격렬하게 변함
- **번개가 튀어나오는 원형 구르기** 등 고유 패턴이 인상적
- 아종: **금오우거** (독 속성)

### 신규 대형 몬스터 (MHP3rd 오리지널)

| 몬스터 | 종별 | 특징 |
|--------|------|------|
| 진오우거 | 팽룡 | 번뢰, 시리즈 간판 |
| 브레이브뉴 | 수룡 | 거대 메기형 수룡 |
| 이블조 | 수룡 | 대형 악어형, 강력한 꼬리 공격 |
| 나바루데우스 | 갑각 | 거대 갑각류, 압도적 등장 연출 |
| 우라간킨 | 암룡 | 드럼통처럼 구르는 거대 갑충 |
| 아그나코톨 | 화룡 | 사막의 마그마형 거대 갑충 |

### MH3에서 이식된 몬스터

| 몬스터 | 특징 |
|--------|------|
| 라기아크루스 | 수중 보스의 육상 이식, 번뢰 속성 |
| 바롯사 | 모래를 파고드는 사막 몬스터 |
| 루드로스 | 물갈퀴 군집 몬스터 |

### 아종 (Subspecies) — 9체 추가

아종은 **하이랭크에서만 등장**합니다. 색이 다르고 속성·패턴이 변형되어 새로운 소재를 드롭합니다.

- 자홍 루드로스 / 청천 우라간킨 / 은화 아그나코톨
- 금오우거 / 흑띠 티그렉스 / 흑이블조 등

### 복귀 인기 몬스터

리오레우스, 리오레이아, 나르가쿠르가, 티그렉스, 디아블로스, 베리오로스, 데빌조 등 시리즈 팬에게 익숙한 몬스터들이 다수 복귀합니다.

## ⚔️ MHP3rd 무기 12종

MHP3rd에는 **스위치 액스**가 신무기로 추가되어 총 **12종**의 무기가 등장합니다.

| 무기 | 특징 | 추천 대상 |
|------|------|-----------|
| **대검** | 최고 위력의 용격베기, 뽑기 공격 | 화력 중시형 |
| **태도** | 정신집중 게이지, 연속 베기 유연 | 밸런스형·입문 추천 |
| **한손검** | 방패 가드 가능, 아이템 사용 중 행동 | 초보 친화 |
| **쌍검** | 귀인화 고속 연타, 이동 자유도 | 기동형 |
| **해머** | 스턴 특화, 머리 집중 타격 | 서포트/딜 |
| **수렵피리** | 연주로 아군 버프, 협력전 핵심 | 멀티 서포트 |
| **랜스** | 가드 성능 최고, 돌진 찌르기 | 방어형 |
| **건랜스** | 포격 기믹 추가, 가드 가능 | 유니크 스타일 |
| **스위치 액스** | **신규.** 도끼↔검 모드 전환 | 변환 전술형 |
| **라이트 보우건** | 속사·이동 특화, 지원탄 가능 | 원거리 기동 |
| **헤비 보우건** | 최고 화력, 거치 플레이 | 원거리 화력 |
| **활** | 차지 샷, 근중거리 대응 | 원거리 기동 |

### 신무기 스위치 액스 (スラッシュアックス)

MHP3rd에서 처음 등장한 무기로, **도끼 모드**와 **검 모드** 사이를 자유롭게 전환하며 싸웁니다.

| 모드 | 특징 |
|------|------|
| **도끼 모드** | 사거리 길고 스태미너 소모 없음, 이동하며 공격 가능 |
| **검 모드** | 근접 고화력, 필살기 **속성해방**으로 병 속성 전부 방출 |

검 모드는 게이지를 소모하므로, 도끼로 게이지를 모으고 → 검 모드에서 화력을 쏟아붓는 리듬이 핵심입니다.

## 🎮 MHP3rd 핵심 시스템

### 아머 스킬 (防具スキル)

방어구마다 **스킬 포인트**가 설정되어 있으며, **같은 스킬의 합계가 +10 이상**이면 해당 스킬이 발동합니다.

| 예시 | 효과 |
|------|------|
| 공격+10 이상 | **공격력UP (소)** 발동 |
| 기절무효+10 이상 | **기절무효** 발동 |
| 회피성능+10 이상 | **회피성능+1** 발동 |

- **부적(탈리스만)**: 채광으로 얻는 악세사리. 스킬과 슬롯을 보유하며 빌드의 핵심
- **장식품**: 방어구의 슬롯에 박아 스킬을 보강

원하는 스킬을 딱 +10에 맞추는 방어구 조합을 고민하는 것이 MH 시리즈 빌드의 재미입니다.

### 아이루(팔리코) 동행 — 2마리로 증가

MHP3rd에서는 처음으로 **아이루 2마리를 동시에 데리고** 퀘스트를 나갈 수 있습니다.

| 항목 | 내용 |
|------|------|
| 역할 | 공격 / 서포트 / 소재 수집 |
| 커스터마이즈 | 무기·방어구·스킬 설정 가능 |
| 효과 | 기절 중 구출, 아이템 사용, 서포트 스킬 발동 |

솔로 플레이 시 아이루 2마리는 **보조 딜러 겸 응급 처치반** 역할을 합니다. 특히 혼자 사냥할 때 기절이나 독 상태에서 아이루가 구해주면 실질적인 생존율이 높아집니다.

### 퀘스트 종류

| 종류 | 내용 |
|------|------|
| 토벌 | 몬스터 1~4체 사냥 |
| 포획 | 약해진 몬스터를 함정+마취구로 생포 (추가 소재 보상) |
| 채집 | 소재·알·광석 수집 |
| 긴급 퀘스트 | 랭크 업을 위한 필수 퀘스트 |

## 📈 랭크 구성 — LR → HR (G랭크 없음)

| 랭크 | 특징 |
|------|------|
| 로우랭크 (LR) | 입문. 몬스터와 기본 패턴 익히기 |
| 하이랭크 (HR) | 강화된 몬스터, 아종 등장, 상위 소재 |
| HR7 | 최고 랭크. 이 작품의 G랭크는 없음 |

G랭크가 없는 것은 MHP3rd의 대표적 단점으로 꼽히지만, **하이랭크까지의 완성도**가 높고 MHP2G보다 쉽게 접근할 수 있어 신규 헌터 유입에 크게 기여했습니다.

## 🇯🇵 일본에서 문화 현상이 된 MHP3rd

MHP3rd는 단순히 잘 팔린 게임이 아니라 **사회 현상**이었습니다.

### 판매 기록

- 발매 첫날: **약 140만 장**
- 발매 2주: **258만 장** (캡콤 역대 최고 초동)
- 누적: **480만 장 이상** (PSP 역대 1위)

### 왜 이렇게 팔렸나

| 이유 | 설명 |
|------|------|
| **PSP 로컬 멀티 문화** | 일본 전철·학교·직장에서 PSP를 꺼내 함께 하는 "집회 플레이" 문화와 완벽한 궁합 |
| **MHP2G의 거대한 팬덤** | 이미 300만 명 이상의 팬베이스가 신작을 기다리고 있었음 |
| **진입 장벽 완화** | G랭크 제거, 아이루 2마리, 온천 체력 강화 등 초보 친화적 개선 |
| **신규 콘텐츠의 질** | 진오우거를 비롯한 매력적인 신규 몬스터들 |
| **대규모 마케팅** | 캡콤의 콜라보·굿즈 캠페인 |

MHP3rd가 팔리던 시기, 일본 전철역 근처 게임샵에는 발매 전날 밤부터 수백 명의 줄이 늘어섰고, 학교에서는 수업이 끝나자마자 PSP를 꺼내 사냥하는 풍경이 일상이 되었습니다.

## 💡 MHP3rd 공략 팁

### 초보자

- **태도 또는 한손검으로 시작**: 태도는 콤보와 이동이 유연하고, 한손검은 아이템을 꺼내면서 움직일 수 있어 초보에게 친화적
- **한 가지 무기에 집중**: 12종 무기를 다 건드리기보다 한 무기를 깊게 파는 것이 성장이 빠름
- **온천 레벨 올리기**: 체력·스태미너 +50은 생존율에 직결
- **포획 퀘스트 활용**: 포획 성공 시 추가 소재 보상이 있어 파밍 효율이 높음
- **아이루 2마리 적극 활용**: 서포트형 아이루를 배치하면 기절 중 구출·소재 수집까지 도움

### 중급자

- **부적 파밍**: 원하는 스킬+슬롯의 부적을 위해 채광 반복. 좋은 부적은 빌드 수준을 크게 올림
- **진오우거는 물 속성 무기로**: 진오우거는 번뢰 속성이라 물 속성이 약점. 초반에 물 속성 무기를 제작해두면 훨씬 수월
- **아종은 속성 대비 필수**: 금오우거(독), 흑띠 티그렉스(무속성+고위력) 등 아종은 원작과 패턴이 달라 별도 준비 필요
- **스위치 액스 도전**: 검 모드 속성해방의 순간 화력이 압도적. 약점 부위에 꽂으면 폭발적인 데미지
- **마을 퀘스트와 길드 퀘스트 병행**: 두 계통에서 각각 긴급 퀘스트 조건이 있어 함께 진행하면 효율적
