**포켓몬스터 초 불가사의 던전(Pokémon Super Mystery Dungeon)**은 스파이크 춘소프트(Spike Chunsoft)가 개발하고 포켓몬 컴퍼니·닌텐도가 퍼블리싱한 닌텐도 3DS용 던전 RPG입니다. 2015년 9월 17일 일본, 2015년 11월 20일 북미에서 발매되었으며, **포켓몬 불가사의 던전 시리즈 최초로 전 세대 720마리 포켓몬이 모두 등장**하는 집대성 작품입니다.

인간이 포켓몬으로 각성한 주인공이 파트너 포켓몬과 함께 탐험대 활동을 하며, 세계를 위협하는 **다크 매터(Dark Matter)**의 음모에 맞서는 스토리가 중심입니다.

## 📖 스토리

### 배경

포켓몬만이 사는 세계. 어느 날 **인간이었던 기억을 가진 주인공**이 포켓몬으로 각성해 눈을 뜹니다. 기억을 잃은 채 새로운 세계를 헤매던 주인공은 파트너 포켓몬을 만나고, 함께 **포켓몬 탐험대(Expedition Society)**에 입단합니다.

### 주요 사건 — 석화 사건

세계 각지에서 포켓몬들이 **돌로 변하는 석화 현상**이 잇따라 발생합니다. 엔테이(Entei)를 비롯한 전설의 포켓몬들도 차례로 석화되어 갑니다.

탐험대는 처음에 동네 불량배 **크루크다일(Krookodile)**을 범인으로 의심하지만, 진짜 흑막은 주인공을 돌봐주었던 **너즐리프(Nuzleaf)**와 전설의 포켓몬 **이벨타르(Yveltal)**였습니다. 너즐리프에 의해 주인공과 파트너도 석화됩니다.

### 공허의 땅 (Voidlands)

석화된 주인공은 **공허의 땅(Voidlands)**이라는 다른 차원에서 눈을 뜹니다. 그곳에서 탐험대 동료들과 재회하고, 세계의 생명 근원인 **생명의 나무(Tree of Life)**가 과거에 **다크 매터(Dark Matter)**에 의해 태양으로 끌려갈 뻔했다는 사실을 알게 됩니다.

### 다크 매터의 정체

다크 매터는 포켓몬들의 부정적인 감정에서 태어난 존재로, 세계 모든 포켓몬을 석화시켜 생명의 나무를 태양으로 끌어당겨 세계를 멸망시키려 합니다. **베헤이이엠(Beheeyem)**은 다크 매터에 빙의된 포켓몬 중 하나로, 주인공의 기억을 지운 장본인이기도 합니다.

### 주요 등장인물

| 캐릭터 | 역할 |
|--------|------|
| **주인공** | 인간에서 각성한 포켓몬. 플레이어가 선택 |
| **파트너** | 활발한 성격의 포켓몬. 주인공의 절친한 동료 |
| **암파로스 (Ampharos)** | 탐험대 대장. 엉뚱하지만 믿음직한 리더 |
| **너즐리프 (Nuzleaf)** | 초반 주인공의 후견인. 사실 흑막의 일원 |
| **메이와 (Mew)** | 과거 주인공을 소환한 신비로운 포켓몬 |
| **다크 매터** | 최종 보스. 세계 멸망을 꾀하는 부정적 에너지의 집합체 |

## 🎮 핵심 시스템

### 주인공·파트너 선택 (20종)

시리즈 1~6세대 스타터 18마리에 **피카츄**와 **리오루**를 더한 **20종**에서 주인공을 선택합니다. 파트너는 주인공을 제외한 나머지 19종 중에서 고릅니다.

| 세대 | 선택 가능 스타터 |
|------|----------------|
| 1세대 | 이상해씨, 파이리, 꼬부기 |
| 2세대 | 치코리타, 브케인, 리아코 |
| 3세대 | 나무지기, 아차모, 물짱이 |
| 4세대 | 모부기, 불꽃숭이, 팽도리 |
| 5세대 | 주리비얀, 찌르꼬, 오소드 |
| 6세대 | 나목령, 펜드라, 개굴닌자 |
| 기타 | **피카츄, 리오루** |

### 턴 기반 던전 탐험

던전 내부는 **랜덤 생성 맵**으로, 매번 구조가 달라집니다. 플레이어가 한 칸 이동할 때마다 던전 내 모든 캐릭터가 동시에 한 행동을 취하는 **완전 턴 기반** 시스템입니다.

| 행동 | 내용 |
|------|------|
| 이동 | 8방향으로 한 칸 이동 |
| 일반 공격 | 전방 1칸 공격 |
| 기술 사용 | PP 소모. 사거리·범위 기술별 상이 |
| 아이템 사용 | 오브·씨앗·식량 등 |
| 대기 | 한 턴 그대로 |

### 배고픔 시스템

던전 탐험 중 **배고픔 게이지**가 점차 감소합니다. 배고픔이 0이 되면 HP가 자동으로 줄어들기 때문에 **식량(Oran Berry, Apple 등) 관리**가 장기 던전 공략의 핵심입니다.

### 커넥션 오브 (Connection Orb) — 모집 시스템

초 불가사의 던전에서 새롭게 도입된 **720마리 전 포켓몬 모집 시스템**입니다.

| 단계 | 내용 |
|------|------|
| 의뢰 수주 | 커넥션 오브에서 포켓몬의 의뢰를 수락 |
| 던전 공략 | 해당 포켓몬이 있는 던전 클리어 |
| 동료 가입 | 조건 달성 시 해당 포켓몬이 동료로 합류 |
| 랭크 보상 | 랭크 업 시 특정 포켓몬 자동 가입 |

**아르세우스**를 동료로 만들려면 전 720마리를 모두 모집해야 하는 궁극의 수집 목표가 존재합니다.

### 포켓몬 파티 구성

던전 입장 시 최대 **4마리** 파티를 편성해 탐험합니다.

| 역할 | 추천 유형 |
|------|----------|
| 딜러 | 공격 기술 특화. 넓은 범위기 보유 포켓몬 |
| 탱커 | 높은 HP·방어. 적의 어그로 분산 |
| 서포터 | 회복기·상태이상기 보유 포켓몬 |
| 전방 | 주인공·파트너 (직접 조작) |

## 🏆 엔딩 후 콘텐츠 (포스트게임)

메인 스토리 클리어 후 **대규모 포스트게임 콘텐츠**가 해금됩니다.

| 콘텐츠 | 내용 |
|--------|------|
| **뮤 서브 스토리** | 탐험대 동료 모집 + 뮤 획득 퀘스트 |
| **전설 포켓몬 모집** | 720마리 수집을 통한 전설·환상 포켓몬 가입 |
| **초고난이도 던전** | 고레벨 전용 던전 개방 |
| **랭크 시스템** | 의뢰 달성으로 랭크 업 → 포켓몬 자동 해금 |

포스트게임 볼륨이 매우 방대해 **메인 스토리보다 포스트게임이 더 길다**는 평가를 받습니다.

## 💡 공략 팁

### 1. 주인공·파트너 선택은 타입 상성 고려

주인공과 파트너의 타입이 겹치지 않도록 선택하면 다양한 던전에 유연하게 대응할 수 있습니다. 예를 들어 불꽃 타입 주인공 + 물·풀 타입 파트너 조합이 안정적입니다.

### 2. 배고픔 관리가 장기 던전의 핵심

식량을 충분히 챙기지 않으면 깊은 층수에서 HP가 자동으로 깎입니다. 사과·큰 사과·바나나 등을 우선적으로 챙기고, 배고픔을 늦추는 아이템도 적극 활용하세요.

### 3. 오브(Orb) 아이템은 위기 시 즉시 사용

오브는 강력한 범위 효과를 발휘하지만 무겁고 귀합니다. 평시에 아끼지 말고, **포위되거나 보스전 직전 위기 상황**에 과감히 사용하는 것이 효과적입니다.

### 4. 기술 PP 고갈 주의

던전이 길수록 기술 PP가 바닥나는 상황이 발생합니다. **PP 회복 아이템(엘릭서류)**을 비상용으로 보관하고, 일반 공격만으로도 처리 가능한 약한 적에게는 기술을 아끼세요.

### 5. 커넥션 오브 의뢰는 꾸준히

메인 스토리 진행 중에도 틈틈이 커넥션 오브 의뢰를 수행하면 동료가 빠르게 늘어나고 랭크도 올라갑니다. 포스트게임에서 720마리 수집에 도전한다면 **스토리 중 의뢰를 미리 쌓아두는 것**이 중요합니다.

### 6. 다크 매터 최종전 대비

최종 보스 다크 매터전은 장기전이 됩니다. 입장 전에 **고급 오브·회복 아이템을 최대한 확보**하고, 파티 포켓몬의 레벨을 충분히 올린 뒤 도전하세요.
